Sass Customization

To set up a Bootstrap 5 project with Sass, install Node.js, run `npm init`, then `npm install bootstrap`. Create a `custom.scss` file to import Bootstrap and override variables like `$primary` or `$border-radius`. Compile with `sass custom.scss custom.css`. Example variables: `$primary: #007bff; $border-radius: 0.5rem;`.
